# Images Directory

This directory contains images used throughout the Outbreak Response Leadership book. Below are instructions for obtaining each required image.

## Required Images

### 1. japanese-wave.jpg (Session 1)

**Purpose**: Metaphor for leadership in times of uncertainty - powerful, dynamic, ever-changing

**Suggested image**: "The Great Wave off Kanagawa" by Katsushika Hokusai
- This iconic Japanese woodblock print (c. 1831) depicts a large wave threatening boats
- Perfect metaphor for navigating powerful, changing forces during outbreak response
- The image is in the public domain

**Where to obtain**:
- [Wikimedia Commons - The Great Wave](https://commons.wikimedia.org/wiki/File:The_Great_Wave_off_Kanagawa.jpg)
- [Metropolitan Museum of Art](https://www.metmuseum.org/)
- Other public domain art repositories

**Image specifications**:
- Format: JPG or PNG
- Recommended resolution: At least 1200px width
- File name must be: `japanese-wave.jpg`

## General Guidelines

### Licensing
All suggested sources (Unsplash, Pexels, Pixabay) offer:
- Free commercial use
- No attribution required (though appreciated)
- High-resolution downloads

### Download Instructions

1. Visit the recommended website
2. Search for the suggested keywords
3. Select an image that matches the description
4. Click "Download" or "Free Download"
5. Choose the highest resolution available
6. Save the file to this `images/` directory
7. Rename it to match the required filename exactly

### Image Optimization

After downloading:
- Ensure images are at least 1200px wide for quality display
- Keep file sizes reasonable (under 2MB if possible)
- JPG format is preferred for photographs
- Maintain good contrast and clarity

### Alternative Images

If you prefer different images:
1. Ensure you have proper rights/licenses to use them
2. Keep the same filename (or update the reference in the corresponding `.qmd` file)
3. Ensure the image conveys the intended metaphor or concept
