# Images Directory

## Required Image

### japanese-wave.jpg

This directory needs an image file named `japanese-wave.jpg` for use in Session 1.

**Suggested image**: "The Great Wave off Kanagawa" by Katsushika Hokusai
- This iconic Japanese woodblock print (c. 1831) depicts a large wave threatening boats
- It's a perfect metaphor for leadership in times of uncertainty
- The image is in the public domain

**Where to obtain**:
- You can download high-quality versions from:
  - [Wikimedia Commons](https://commons.wikimedia.org/wiki/File:The_Great_Wave_off_Kanagawa.jpg)
  - [Metropolitan Museum of Art](https://www.metmuseum.org/)
  - Other public domain art repositories

**Image specifications**:
- Format: JPG or PNG
- Recommended resolution: At least 1200px width for good quality display
- File name must be: `japanese-wave.jpg`

**Copyright**: Ensure the image is in the public domain or you have proper rights to use it. The original Hokusai print is in the public domain.

## Alternative

If you prefer a different image representing waves or uncertainty, you can use any appropriate image, but make sure to:
1. Name it `japanese-wave.jpg` (or update the reference in `session-01-leadership-meaning.qmd`)
2. Ensure you have rights to use the image
3. Choose an image that represents the metaphor of navigating powerful, changing forces
